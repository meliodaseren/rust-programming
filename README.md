# Rust tutorial

## Installation

### Linux or macOS

```shell
$ curl https://sh.rustup.rs -sSf | 
$ source $HOME/.cargo/env
```

### Update

```shell
$ rustup update
```

### Remove

```shell
$ rustup self uninstall
```

### Troubleshooting

```shell
$ rustc --version
```

## Compile

```shell
$ rustc main.rs
$ ./main
```
